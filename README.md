# FeatherCanvas Studio

FeatherCanvas Studio 是一个基于 Flutter 的 AI 生图客户端骨架。
目标很直接：让用户自己填写图片生成 API、提示词和参数，
然后快速生成、预览和管理图片结果。

## 当前定位

- API 配置入口
- 提示词编辑
- 生成参数面板
- 图片结果与历史记录

## 计划支持

- OpenAI Images
- Stability AI
- ComfyUI 本地接口
- AUTOMATIC1111 本地接口

## 本地开发

```bash
flutter pub get
flutter run -d windows
```
